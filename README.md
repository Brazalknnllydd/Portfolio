# Portfolio

Personal portfolio built with React, TypeScript, and Vite.

## Features

- Single-page portfolio experience
- Smooth section navigation
- Dark and light mode toggle
- Contact modal powered by EmailJS
- Responsive layout for desktop and mobile

## Tech Stack

- React
- TypeScript
- Vite
- Framer Motion
- Tailwind CSS
- shadcn-style UI components

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
